# Доступ в кабинет администратора

Административный кабинет вынесен из кабинета пользователя и доступен по отдельной закрытой ссылке:

```text
/admin
```

Для локального запуска:

```bash
npm run dev --workspace @med-analiz/frontend
```

После запуска откройте:

```text
http://localhost:3000/admin
```

Кабинет пациента находится отдельно:

```text
http://localhost:3000/app
```

Главная страница до регистрации находится по корневому адресу:

```text
http://localhost:3000/
```

В текущем прототипе авторизация и права доступа представлены mock-интерфейсом. При подключении реального backend нужно ограничить `/admin` проверкой роли администратора.
